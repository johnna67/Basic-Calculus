# Infinite Loop: No Escape

A desktop-first, first-person calculus horror escape game. Explore five rooms, survive the Entity, solve 17 questions covering Basic Calculus Lessons 1–12, and reach one of three endings.

**[Play the game](https://ajegz.github.io/infinite-loop-no-escape/)**

## Features

- Five playable rooms and a cumulative Final Exam
- Hints, explanations, and complete worked solutions
- Scoring, timer, Entity pressure, spirit choices, and achievements
- Keyboard and mouse controls with accessibility settings
- Local autosave with no account or backend required

## Run locally

Requires Node.js 22.12 or later.

```bash
npm ci
npm run dev
```

To verify and build:

```bash
npm run check
npm run build
```

Built with TypeScript, Phaser, Vite, and KaTeX. See the [documentation index](docs/00_DOCUMENTATION_INDEX.md) for the complete design, mechanics, answer key, testing, and deployment notes.
