# Ready-to-Paste Initial Codex Task

Read `AGENTS.md`, `CODEX_START_HERE.md`, `HANDOFF_STATUS.md`, and the documentation index before editing anything. Treat this folder as the project root.

First:

1. Inspect the repository structure.
2. Run `npm ci`.
3. Run `npm run check`.
4. Run `npm run build`.
5. Report any failure with the exact command and relevant error output.

Then review the implementation against the non-negotiable requirements in `AGENTS.md` and the original proposal in `reference/BASCALC_Culminating_Project_Proposal.pdf`. Preserve the static GitHub Pages architecture, five-room progression, seventeen-question curriculum, full keyboard/mouse parity, educational solutions, save system, accessibility settings, and multiple endings.

Do not begin a broad rewrite. Summarize the current architecture, identify the highest-value improvement or defect, propose a focused plan, and wait for approval before changing several systems at once. For any approved code change, update tests and documentation, then rerun `npm run check` and `npm run build`.
